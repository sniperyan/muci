## 更新日志

### 1.0.15

`2018-07-26`
* 移除对saas的支持，纯属安装依赖速度考虑，因为项目不用而每次引入sass的依赖太慢，如需使用saas可手动引入`sass-loader`和`node-sass`

### 1.0.6

`2018-5-26`

* less 版本需要指定 2.7.2 ,不要使用3.x.x版本，否则antd在windows上会报错
* 尽可能的保持`antd`的设计语言。
